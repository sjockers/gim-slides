Ein einfaches HTML-Dokument

					<!doctype html>
					<html>
					  <head>
					    <title>Das ist der Titel der Seite</title>
					  </head>
					  <body>
					    <h1>Das ist die Überschrift</h1>
					    <p>Das ist etwas Fließtext.</p>
					  </body>
					</html>

Doctype Declaration

Old School: XHTML1 
					<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
						"http://www.w3.org/TR/html4/strict.dtd">
New School: HTML5 
					<!doctype html>

Head: Meta-Informationen

					<!doctype html>
					<html>
					  <head>
					    <title>Das ist der Titel der Seite </title>
					    <link rel="stylesheet" href="style.css" />
					  </head>
					(...)

Body: Inhalt des Dokuments

					(...)
					  <body>
					    <h1>Das ist der Titel der Seite</h1>
					    <p>Das ist etwas Fließtext.</p>
					  </body>
					</html>

Texte formatieren (Beispiele)

  • <h1> </h1> – Überschriften (h1 bis h6)
  • <p> </p> – Absätze
  • <em> </em> – Betonung (Emphasis)
  • <strong> </strong> – Starke Betonung

Sonstige Elemente (Beispiele)

  • <table> </table> – Tabellen
  • <ul> </ul> / <ol> </ol> – Listen
  • <a href="..."> </a> – Links (Anchor)
  • <img src="..." alt="..." /> – Bilder

HTML-Dokumente strukturieren (Beispiel)

					<body>
					  <header>(...)</header>
					  <section>(...)</section>
					  <aside>(...)</aside>
					  <footer>(...)</footer>
					</body>

HTML-Dokumente strukturieren (Beispiel)

					<body>
					  <header>(...)</header>
					  <section>(...)</section>
					  <aside>(...)</aside>
					  <footer>(...)</footer>
					</body>

Achtung: Die Verwendung dieser Elemente macht keine Aussage über die Darstellung des Dokuments im Browser. Für das visuelle Anordnen der Elemente wird CSS verwendet.

HTML-Dokumente strukturieren: Weitere Elemente

  • <article> </article> – Artikel
  • <nav> </nav> – Navigation
  • <figure> </figure> – Abbildung
  • <figcaption> </figcaption> – Bildunterschrift
  • <div> </div> – Abschnitt (keine besondere Semantik)
Foto: Billy Abbott

Darstellung vs. Inhalt

  • Structure – HTML
    • logische Strukturierung des Inhalts
  • Presentation – CSS
    • visuelle Darstellung
  • Behavior – JavaScript
    • Interaktion mit dem Nutzer

Progressive Enhancement

						body {
							font-family : sans-serif;
						}
						h1 {
							color : green;
						}
						p {
							padding : 20px;
							background-color : black;
							color : white; 
						}
