Einbinden von CSS

In der Regel als externes Dokument:
<link rel="stylesheet" href="style.css" />
Alternative: Inline:
<div style="color: blue;">...</div>
Alternative: Als eigenständiger HTML-Tag:
<style>div {color: blue;}</style>

Definieren von CSS-Regeln

          selector { 
            property1: value; 
            property2: value; 
            property3: value; 
          }

Whitespace

Folgende Code-Beispiele haben das gleiche Ergebnis:

          p { 
            font-family: arial; 
            color: blue; 
          } 
        
          p {font-family: arial; color: blue;}

Verschiedene Typen von Selektoren (Beispiele)

Der Element-Selektor:
div {color: blue;}
Der Class-Selektor:
.content {color: blue;}
Der ID-Selektor:
#main {color: blue;}

Gruppieren von Selektoren

Folgende Code-Beispiele haben das gleiche Ergebnis: 

          p {color: blue;} 
          h1 {color: blue;} 
          #example {color: blue;} 
        
          p, h1, #example {color: blue;}

Kombinieren von Selektoren (Beispiele)

          p.warning { 
             /* p-Elemente mit der Klasse warning */ 
          }           
        
          div#example { 
             /* div-Elemente mit der ID example */         
        }

Kombinieren von Selektoren (Beispiele)

          p .warning { 
             /* Elemente mit der Klasse warning 
                innerhalb eines p-Elements */ 
          }  
        
         
          .warning p { 
             /* p-Elemente innerhalb von
                Elementen mit der Klasse warning */ 
          }

Kombinieren von Selektoren (Beispiele)

          div#example p em.warning { 
             /* em-Elemente mit der Klasse warning 
                innerhalb von p-Elementen 
                innerhalb eines div-Elements 
                mit der ID example */ 
          }

Layout mit CSS

  1. Das Box-Modell
  2. Die Float-Eigenschaft
  3. Die Position-Eigenschaft

Das Box-Modell

  • CSS-Eigenschaften:
    • display: inline; – Element fließt im Text
    • display: block; – Element bildet eine "Box"
    • etc.

Details: https://developer.mozilla.org/en/CSS/display

Abbildung: Mozilla Foundation

Fluss von Elementen

  • CSS-Eigenschaften:
    • float: left; – Elemente "fließen" nach links
    • float: right; – Elemente "fließen" nach rechts
    • clear: left; – Aufheben der Float-Eigenschaft
      (auch clear: right; und clear: both;)
    • etc.

Details: https://developer.mozilla.org/en/CSS/float

Positionierung von Elementen

  • CSS-Eigenschaften:
    • position: relative; – Relativ zur eigenen Position
    • position: absolute; – Absolut zum Elternelement
    • top: 100px; – Offset (auch left, right, bottom)
    • etc.

Details: https://developer.mozilla.org/en/CSS/position
