Was ist Farbe?

  • Sichtbare elektromagnetische Strahlung — Licht
    • Farbwahrnehmung: zwischen 380 nm und 780 nm
  • Wahrgenommen durch Rezeptoren auf der Netzhaut:
    • Stäbchen (Hell-/Dunkel-Kontrast)
    • Zapfen (Farbe)
  • Farbwahrnehmung variiert
    • Farbfehlsichtigkeit (z.B. Rotschwäche, Grünschwäche)

Farbe im elektromagnetischen Spektrum

Rot ~ 790–630 nm ~ 379–476 THz
Orange ~ 630–580 nm ~ 476–517 THz
Gelb ~ 580–560 nm ~ 517–535 THz
Grün ~ 560–480 nm ~ 535–624 THz
Blau ~ 480–420 nm ~ 624–714 THz
Violett ~ 420–390 nm ~ 714–769 THz

Kulturelle Bedeutung

  • Assoziationen: z.B. Grün (Gras, Natur), Blau (Meer, Himmel)
  • Symbolik: z.B. Grün (Hoffnung), Rot (Liebe), Weiß (Unschuld)
  • Abhängig vom Kontext!

Subtraktive und additive Farbmischung

Bildschirmdarstellung

  • Pixel: "Picture Element"
    • "atomare Einheit" der Bildschirmdarstellung
    • ein Pixel besteht aus drei Komponenten (Subpixel): Rot, Grün, Blau
  • additive Farbmischung!
Abbildung: Wikipedia

Das RGB-Farbsystem

  • drei Komponenten: Rot, Grün, Blau
  • analog zur tatsächlichen Farberzeugung auf dem Bildschirm
  • Beispiel: Reines Rot
    • Dezimale Darstellung: 255, 0, 0
    • Hexadezimale Darstellung: #FF0000

Das HSL-Farbsystem

  • Beschreibung von Farbe über drei Komponenten:
    1. Hue (Farbton)
    2. Saturation (Sättigung)
    3. Lightness; alternativ: Brightness, Luminance (Helligkeit)
  • ähnelt der menschlichen Farbwahrnehmung
  • intuitiver verwendbar als RGB
  • Beispiel: Reines Rot 0, 100%, 50%

Exkurs: Gängige Grafikformate im Web

Unterscheidungsmerkmale für Grafikformate:
  • Dateigröße
  • Kompressionsverfahren
  • Farbtiefe (Anzahl der darstellbaren Farben)
  • Alphakanal (Transparenz)

Grafiken mit hoher Farbtiefe (z.B. Fotos)

  • in der Regel 3 Farbkanäle (RGB) mit je 8 bit
  • JPEG
    • verlustbehaftete Kompression
    • kein Alphakanal
  • PNG ("PNG 24")
    • verlustfreie Kompression
    • optionaler Alphakanal (in der Regel 8 bit)

Grafiken mit geringer Farbtiefe (z.B. Symbole)

  • bis zu 256 Farben (in der Regel selektiv)
  • verlustfreie Kompression (aber oft Reduktion der Farbtiefe)
  • GIF
    • 1 bit Alpha
    • oft für Animation verwendet (Beispiel)
  • PNG ("PNG 8")
    • Alpha > 1 bit möglich

Farbe & CSS

Anwendungsmöglichkeiten:
  • Hintergrundfarbe: background-color
  • Schriftfarbe: color
  • Rahmenfarbe: border-color

Farbangaben in CSS

  • Farbnamen, z.B. red
  • Numerische Angaben (hexadezimal), z.B. #FF0000
  • Numerische Angaben (dezimal), z.B. rgb(255, 0, 0)
  • Numerische Angaben (prozentual), z.B. rgb(100%, 0, 0)

Nachlesen, z.B. hier: https://developer.mozilla.org/en/CSS/color_value

Farbangaben in CSS (advanced)

  • HSL, z.B. hsl(0, 100%, 50%)
  • HSLA (inklusive Transparenz), z.B. hsla(0, 100%, 50%, 0.7)
  • RGBA (dezimal, inklusive Transparenz), z.B. rgba(255, 0, 0, 0.7)

Nachlesen, z.B. hier: https://developer.mozilla.org/en/CSS/color_value
