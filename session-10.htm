Warum JavaScript-Bibliotheken?

JavaScript-Programmierung im Browser ist nicht besonders effizient

  • Arbeiten mit dem DOM ist umständlich
    (sehr viel Code notwendig, auch für einfache Manipulationen)
  • JavaScript-Implementierung im Browser ist inkonsistent
    (Anpassungen für verschiedene Browser notwendig)

Warum JavaScript-Bibliotheken?

JavaScript-Bibliotheken...

  • lösen die beschriebenen Probleme
  • bieten zusätzliche Funktionalität
  • vereinfachen und beschleunigen die Entwicklung

Beispiel: jQuery

Funktionalität

  • DOM-Manipulation
  • Event-Handling
  • Ajax
  • Animationen und Effekte
  • Funktionalität kann durch Plugins erweitert werden.

jQuery ist eine einzelne JavaScript-Datei, die zusätzlich zu den eigenen Scripts eingebunden wird. 

				 <!doctype html>
				  <html> 
				    <head> 
				      <script src="jquery.js"></script> 
				      <script src="script.js"></script>
				    </head> 
				    (...)

jQuery: Grundlegende Funktionsweise

$("li").addClass("photo");

  1. Finde Elemente im DOM.
  2. Tu etwas mit diesen Elementen.

Das Dollar-Zeichen ist ein Platzhalter für jQuery .

				Diese Zeile:
				$("li").addClass("photo");
				
... enstpricht dieser Zeile:
				jQuery("li").addClass("photo");

Die Auswahl der DOM-Elemente erfolgt durch CSS-Selektoren.

				$("li")
				$(".some-class")
				$("#some-id")
				$(".some-class img")
				$("#some-id > input[type="radio"]")

jQuery umfasst eine Vielzahl von Hilfsfunktionen für die Manipulation von Elementen. Hier einige Beispiele:

				$("li").addClass("some-class");
				$("li").removeClass("some-class");
				$("li").show();
				$("li").hide();
				$("li").append(someElement);
				$("li").remove(someElement);

Zusammenfassung 

				

					$("li").addClass("photo");
				

				

					$("li")
					... finde alle li-Elemente im aktuellen Dokument.
				

				

					addClass("photo")
					... füge allen gefundenen Elementen die Klasse „photo“ hinzu.
					
(Wenn keine Elemente gefunden werden, passiert nichts)

Event-Handling mit jQuery

bind kapselt die Eigenheiten verschiedener Browser in eine einfache, einheitliche API. 

				$('#foo').bind('click', function() {
				  // do something when #foo is clicked
				});

Beispiel: Event-Handling mit jQuery 

				

					$('a').bind('click', someFunction)
					... delegiert den Click-Event für alle a-Elemente des Dokuments						
				

				

					$('a').unbind('click', someFunction)
					... entfernt den Event-Handler wieder von allen a-Elementen

Besonderheit: Der Document-Ready-Event 

				

					window.onload = function() {
					  // this code is run when the website is loaded 
					  // (including images, etc.)
					};
				


				

					$(document).ready(function() {
					  // this code is run when the DOM is loaded
					  // (without images, etc. – much faster!)					
					});

Asynchrone Kommunikation mit jQuery

  • $.ajax-API in jQuery
  • kapselt die Eigenheiten verschiedener Browser in eine einfache API
  • integriert auch verwandte Techniken wie JSONP

Beispiel: AJAX mit jQuery 

				$.ajax({
				  url: "/data/",
				  success: function(data){
				    $("#results").append(data);
				  }
				});

Beispiel: JSONP mit jQuery 

				$.ajax({
				  url: "http://some-domain.com/jsonp/",
				  dataType: 'jsonp',
				  success: function(data){
				    $("#results").append(data);
				  }
				});

Exkurs: Sichtbarkeit von Variablen in JavaScript

  • lokale Variablen (functional scope)
    • sichtbar in der Funktion in der sie deklariert wurden
  • globale Variablen (global scope) – Eigenschaften des window-Objekts
    • Variablen die außerhalb einer Funktion deklariert wurden
    • Variablen die nicht explizit deklariert wurden

Das IIFE-Pattern

				(function(){
				  // local scope
				})();
  • IIFE – Immediately-Invoked Function Expression
  • erzeugt lokalen Scope innerhalb einer anonymen Funktion 
				(function($){
				  
				  // private code goes here:
				  // (...)
				  
				  // make functionality public through the jQuery-object:
				  $.fn.myJQueryPlugin = function() {
				    // (...)
				  }
				})(jQuery);
